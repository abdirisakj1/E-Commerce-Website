# 🛍️ Utisil Products – E-Commerce Website

Uttisil Products is responsive E-Commerce website built using HTML, CSS, and JavaScript.  
This project demonstrates including product listing, shopping cart functionality, real-time price updates, and local storage integration.

---
## 🚀 Features

- Home page (Landing page)
- About page
- Shop page with product listing
- Contact page
- Add to Cart functionality
- Increase and decrease quantity
- Real-time total price calculation
- Local Storage to save cart data
- Responsive design (Mobile & Desktop)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Browser Local Storage API

---

## 🛒 Shopping Cart System

- Users can add products to the cart.
- Quantity can be increased or decreased.
- Total amount updates automatically.
- Cart data is stored in the browser using localStorage.
- Cart remains saved even after refreshing the page.

---

## 💡 How It Works

1. User visits the Shop page.
2. Selects a product and clicks "Add to Cart".
3. Cart updates instantly.
4. User can increase or decrease quantity.
5. Total price recalculates in real time.
6. Cart data is saved in local storage.
